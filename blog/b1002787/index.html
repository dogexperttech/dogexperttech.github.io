Fixing AMD Driver Crashes

August 24, 2025 | By Dog Expert Tech

⚠️ UPDATE: DECEMBER 12, 2025
If you are still experiencing driver timeouts, please update your Motherboard BIOS to the latest version. This is critical for system stability.

If you have an AMD system, you've likely seen this: the screen freezes, goes black, and a driver timeout error appears. This is often followed by the "Code 31" error in Device Manager.

1. Driver Cleanup & Reinstall

The first step is a complete cleanup of old drivers.

  • Use DDU (Display Driver Uninstaller) in Safe Mode to wipe all GPU driver remnants.
  • Download DDU Here
  • Reinstall the latest stable drivers from the official AMD Support Page.

2. Disable Hyper-V & VBS

Windows features like VBS and Memory Integrity can conflict with AMD drivers, causing black screens and system hangs.

3. Automated Fix Script

📥 Download disable-hyperv.bat

Right-click the downloaded file and select "Run as Administrator"

@echo off
echo ===========================
echo Hyper-V / VBS Killer Script
echo ===========================

:: 1. Disable Hyper-V and Virtual Platforms
echo [1/5] Disable Hyper-V and platforms...
dism /Online /Disable-Feature:Microsoft-Hyper-V-All /NoRestart
dism /Online /Disable-Feature:HypervisorPlatform /NoRestart
dism /Online /Disable-Feature:VirtualMachinePlatform /NoRestart
dism /Online /Disable-Feature:WindowsSubsystemForLinux /NoRestart
dism /Online /Disable-Feature:Windows-Defender-ApplicationGuard /NoRestart

:: 2. Disable VBS (Virtualization Based Security)
echo [2/5] Disabling VBS...
reg add "HKLM\System\CurrentControlSet\Control\DeviceGuard" /v EnableVirtualizationBasedSecurity /t REG_DWORD /d 0 /f
reg add "HKLM\System\CurrentControlSet\Control\DeviceGuard" /v RequirePlatformSecurityFeatures /t REG_DWORD /d 0 /f

:: 3. Disable Credential Guard
echo [3/5] Disabling Credential Guard...
reg add "HKLM\System\CurrentControlSet\Control\Lsa" /v LsaCfgFlags /t REG_DWORD /d 0 /f

:: 4. Disable Memory Integrity (Core Isolation)
echo [4/5] Disable Memory Integrity...
reg add "HKLM\SYSTEM\CurrentControlSet\Control\DeviceGuard\Scenarios\HypervisorEnforcedCodeIntegrity" /v Enabled /t REG_DWORD /d 0 /f

:: 5. Disable hypervisor startup
echo [5/5] Disable hypervisor launchtype...
bcdedit /set hypervisorlaunchtype off

echo ========================================
echo Done! Your PC will restart in 10 seconds.
echo Save your work now!
echo ========================================
timeout /t 10
shutdown /r /t 0
