Fixing AMD Driver Crashes

August 24, 2025 | By Dog Expert Tech

Note: Updating your motherboard BIOS is often the best first step before adjusting drivers.

If your AMD system is freezing, blacking out, or showing driver timeout errors, the next step is a clean driver reinstall and disabling the Windows features that can conflict with AMD GPU drivers.

1. Driver Cleanup & Reinstall

Use DDU in Safe Mode to completely remove old GPU drivers. Then install the latest official AMD drivers for your GPU.

  • Download DDU Here
  • Run DDU in Safe Mode.
  • Reinstall the latest stable drivers from the official AMD Support Page.

2. Disable Hyper-V & VBS

Virtualization features such as Hyper-V and Virtualization-Based Security can cause driver instability with AMD GPUs.

3. Use the automated fix script

📥 Download disable-hyperv.bat

Right-click the downloaded file and choose "Run as Administrator".

@echo off

echo ===========================
echo Hyper-V / VBS Killer Script
echo ===========================

dism /Online /Disable-Feature:Microsoft-Hyper-V-All /NoRestart
dism /Online /Disable-Feature:HypervisorPlatform /NoRestart
dism /Online /Disable-Feature:VirtualMachinePlatform /NoRestart
dism /Online /Disable-Feature:WindowsSubsystemForLinux /NoRestart
dism /Online /Disable-Feature:Windows-Defender-ApplicationGuard /NoRestart

reg add "HKLM\System\CurrentControlSet\Control\DeviceGuard" /v EnableVirtualizationBasedSecurity /t REG_DWORD /d 0 /f
exit /b
